🚀 HOW TO HOST & MAKE REAL-TIME
1
Create Firebase Project (Free)
Go to console.firebase.google.com → New Project → Add a Web App → Copy the firebaseConfig object and paste it into this HTML file replacing the placeholder config at the top of the script section.
2
Enable Realtime Database
In Firebase Console → Build → Realtime Database → Create Database → Choose a region → Start in Test Mode (allows all reads/writes for 30 days, then set rules).
3
Host on Netlify (Easiest — Free)
Go to netlify.com → Sign up free → Drag & drop this HTML file into the deploy box → Get a free link like your-site.netlify.app — share it with everyone!
4
Host on GitHub Pages (Free)
Create a GitHub repo → Upload this file as index.html → Go to Settings → Pages → Source: main branch → Your site is live at username.github.io/repo
5
Use Admin Panel to Add Streams
Click Admin in the navbar → Password: BA56ULTRA → Paste your YouTube/Twitch/M3U8 links → Click Save → All users on your site see the streams instantly!
6
Custom Domain (Optional)
On Netlify, go to Domain Settings → Add custom domain. On Firebase Hosting, run firebase deploy after firebase init hosting. Point your domain's DNS to the provided records.
